# UCL Biosciences Computational Support

**Welcome!** This site guides you through UCL's computational resources, data storage, HPC systems, and training opportunities.

> **Who is this for?** Researchers at UCL Biosciences whose data are too large to store locally, whose analyses need more compute power, or who need help with data sharing and management.

## Quick Facts

| Resource | Free? | Best for |
|---|---|---|
| **Myriad HPC** | ✅ UCL staff/students | General biosciences workloads, moderate parallelism |
| **Kathleen HPC** | ✅ UCL staff/students | Large parallel jobs, high core counts |
| **RDSS** | ✅ 1TB free | Active research data, team collaboration |
| **RDR** | ✅ | Published/archived datasets (50GB default) |
| **OneDrive** | ✅ | Lightweight documents, Office files |
| **Globus** | ✅ | Large file transfers (>10GB), HPC transfers |
